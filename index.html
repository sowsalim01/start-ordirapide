Comment utiliser ce programme

  1. Remplacez l'URL dans le code JavaScript (variable RENDER_SITE_URL) par l'URL de votre site Render
  2. Déployez cette page sur GitHub Pages en poussant le code dans un dépôt GitHub
  3. Ouvrez la page GitHub Pages dans votre navigateur et laissez l'onglet ouvert
  4. Le script enverra automatiquement une requête à votre site Render toutes les 5 minutes

Ce programme fonctionne entièrement côté client et ne nécessite pas de backend

Les requêtes sont envoyées depuis votre navigateur via l'API Fetch
